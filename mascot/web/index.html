How this was made

One 332 × 301 reference image was measured, reconstructed as smooth parametric forms in Blender, then calibrated against repeated renders.

  1. 01
    Measure

    Extract silhouette, feature bounds, spacing, and exact background color from the source pixels.

  2. 02
    Model

    Build the shell and display as superellipsoids, sculpt the ears into the crown, then add eyes, smile, and detached side pods.

  3. 03
    Match

    Use an orthographic camera and broad studio lights. Render, overlay, measure error, and adjust one controlled parameter at a time.

  4. 04
    Translate

    Export real geometry to glTF and reproduce the view-dependent Blender materials in Three.js so highlights still move when you orbit.

  5. 05
    Animate

    Layer idle, gaze, expressions, and gestures on separate pivots, then use velocity-preserving springs so rapid input redirects without snapping.

  6. 06
    Sound

    Generate a restrained ambient bed and gesture cues, master them as static assets, then trigger each cue only when its animation truly starts.
